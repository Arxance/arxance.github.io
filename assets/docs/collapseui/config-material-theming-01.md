### Material UI Theming System
This is a **custom user template theme** for **Linux with Noctalia Shell** (for now) that changes the Collapse-UI Theme System Dynamically that fully depends on wallpaper.
<br><br>
The way it works is - Detects whenever the wallpaper changed, Fetch and extract wallpaper colors to create a cohesive theme for Collapse-UI and Apply it.
<br><br>
However, Bedrock Edition does not reload pack once the UI theme has been applied, so you will have to `/reload all` if you are in the world with cheat on. or leaving the world and join, or restart the game as a whole.
<br><br>
This will only replace a default theme configuration, so you can keep a modified certain config as you will!
<br><br>
> **Requirements**<br>
For now, this might change in the future if someone decides to port it to windows or android.

For Linux
- Package `Python3` are required. you should, well. already have them if you compile some of apps.
- `Noctalia Shell` are required. it's a current supported shell at the moment. It should support Niri and Hyprland too.
- A Text Editor such as `Kate`, Will be required to modify certain files. but should not affect the shell update as a whole.
